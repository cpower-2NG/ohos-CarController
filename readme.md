# CarControler

开源鸿蒙手机端平衡车控制应用。当前仓库用于实验八的前端应用开发，并配套嵌入式控制代码。

## 项目目标

面向平衡车控制场景，完成一个鸿蒙手机前端应用，后续可逐步接入 TCP、星闪、AI 对话、语音识别和地图导航能力。

## 预期功能

### 1. 车辆运动控制

- 连接平衡车服务端
- 发送前进、后退、左转、右转、停止指令
- 显示当前连接状态
- 支持速度调节

### 2. AI 智能对话

- 输入文本后接入大模型
- 将自然语言解析为控制指令
- 支持车灯、座椅、氛围灯等扩展控制

### 3. 语音识别

- 将语音转为文字
- 支持实时语音输入
- 为后续 AI 控制做入口

### 4. 地图导航

- 预留地图页面
- 支持定位、路线规划、导航展示

### 5. 设置与管理

- 服务端地址配置
- 调试参数配置
- 后续扩展账号、设备、日志管理
